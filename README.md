# zuo-cli-hello-world

## Description
A POC for zuo programming language.

## Tech stack
- gcc
- make

## Docker stack
- ubuntu:22.04

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
